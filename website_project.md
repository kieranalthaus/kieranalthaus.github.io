# Project Overview: Personal Academic Website

## Objective
Create a static personal website to be hosted on GitHub Pages. The site will serve as a professional portfolio highlighting my research in population genetics, phylogenetics, biogeography, speciation, and evolutionary biology broadly. 

## Directory Structure
The workspace is organized as follows:

* `/` (Root): The main project directory where the final compiled site files (e.g., `index.html`, CSS, and JS) should be generated.
* `/template/`: Contains the baseline templates to be used for the site's layout and structural foundation.
* `/CV/`: Contains my CV. This document is the primary data source for the website's text content, including my experience, publications, skills, and research projects.

## Design Preferences
* **Aesthetic:** Minimalist, clean, and professional. The focus should remain heavily on readability and content hierarchy. Use the CSS style website provided in the template directory.
* **Color Palette:** The primary theme should be neutral (e.g., white/off-white backgrounds with dark, highly legible text). However, please incorporate primary colors (red, blue, yellow) as tasteful accents. Use these primary colors for interactive elements like links, buttons, hover states, or subtle section dividers to add a bit of character without cluttering the minimal layout.

## Instructions for ClaudeCode
1. **Analyze Content:** Read the CV located in the `/CV/` directory to extract my professional background, technical skills, and research focus.
2. **Review Templates:** Examine the structural files in the `/template/` directory to understand the desired starting point.
3. **Synthesize & Generate:** Build the website in the root directory by intelligently mapping the extracted CV content into the provided templates. Ensure the copy is formatted well for the web.
4. **Style:** Apply the requested minimal design with primary color accents via CSS, ensuring the site is responsive and ready for GitHub Pages deployment.
